#  Sales Data Analysis using Python

##  Overview
This project analyzes e-commerce sales data to uncover trends, patterns, and business insights.

##  Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

##  Dataset
The dataset contains:
- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit

##  Analysis Performed
- Data Cleaning
- Sales & Profit Calculation
- Category-wise Analysis
- Region-wise Analysis
- Monthly Sales Trends

##  Visualizations
- Bar Charts
- Line Charts
- Pie Charts
