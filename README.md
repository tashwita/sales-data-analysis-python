#  Sales Data Analysis using Python

##  Overview
This project analyzes e-commerce sales data to uncover trends, patterns, and business insights.

##  Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

##  Dataset
The dataset contains:
- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit

##  Analysis Performed
- Data Cleaning
- Sales & Profit Calculation
- Category-wise Analysis
- Region-wise Analysis
- Monthly Sales Trends

##  Visualizations
- Bar Charts
- Line Charts
- Pie Charts

## Key Insights
- Electronics category generates highest sales
- North & South regions show strong performance
- Sales vary across months indicating seasonal trends

##  Future Improvements
- Add Machine Learning model for sales prediction
- Build interactive dashboard using Streamlit
